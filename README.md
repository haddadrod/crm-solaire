# CRM Solaire

Mon CRM pour mon activité de panneaux solaires.

## 🚀 Mise en ligne

Ce projet utilise :
- **Vite + React** pour le frontend
- **Supabase** pour la base de données et l'authentification
- **Tailwind CSS** pour le style

## 📦 Installation locale (optionnel)

Si tu veux faire tourner le projet sur ton ordinateur :

```bash
npm install
npm run dev
```

## 🚀 Déploiement sur Vercel

1. Push ce code sur GitHub
2. Connecte Vercel à GitHub
3. Importe le projet
4. Ajoute les variables d'environnement :
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
5. Déploie !

## 🗄️ Configuration Supabase

Voir le fichier `SUPABASE_SETUP.md` pour les étapes de création de la base de données.
